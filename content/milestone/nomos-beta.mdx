# Nomos Public Beta

Opening the audit trail platform to external AI teams.

## Scope

- Public API documentation
- Self-serve domain creation
- Dashboard for viewing decisions
- Export for compliance audits
- SDK for popular languages

## Success Criteria

- 20+ external beta users
- 1M+ events logged per month
- SDK adoption in 3+ languages
- Clear compliance use case validated

## Target Users

- AI teams building autonomous agents
- Compliance teams needing audit trails
- MLOps teams tracking model decisions
