Convex's reactive document database with automatic real-time sync.

## Features

- Document-based data model
- Automatic real-time subscriptions
- ACID transactions
- TypeScript-first schema definitions
- Built-in full-text search

## Comparison to Firestore

Similar document model to Firestore but with:
- Stronger typing via TypeScript schemas
- Built-in reactivity (no manual listeners)
- Simpler query model
- US-only deployment (vs Firestore's multi-region)
