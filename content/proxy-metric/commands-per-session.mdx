# Commands Per Session

Average number of meaningful commands executed per SmartBox session.

## Why This Metric

High command counts indicate the product is useful enough to keep using. Low counts suggest friction or lack of value. This is a leading indicator of retention.

## Current State

Product not yet at measurement stage.

## Target

15+ commands per session on average.

## What Counts

- Actual work commands (create, edit, run, deploy)
- Not navigation or settings
- Not failed commands or retries
